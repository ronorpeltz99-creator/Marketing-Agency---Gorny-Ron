# Workflow: Shopify Store Setup & Product Import

This workflow guides an agent through the process of setting up a new store and importing a winning product.

## Steps

1. **Create Store**
   - Use the `createStore` tool with the requested brand name.
   - Wait for store URL and credentials.

2. **Prepare Product Data**
   - Analyze the product from AliExpress/DSers.
   - Rewrite the title and description for high conversion (Marketing Agency style).

3. **Import Product**
   - Use the `importProduct` tool to push the data to the newly created store.
   - Ensure high-quality images are included.

4. **Notification**
   - Use the `sendAlert` tool to notify the owner that the store is ready for review.

## Success Criteria
- Store URL is accessible.
- Product is visible in the store admin.
- Owner received a Telegram/Discord notification.
